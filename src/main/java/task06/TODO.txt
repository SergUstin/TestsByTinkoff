Дан массив a из n целых чисел.
Требуется выполнить q запросов такого вида (1≤l≤r≤n,0≤k,b,x≤109):
lrx — прибавить x ко всем ai на отрезке i∈[l,r]?lrkb — вывести max l≤i≤r min(ai,k⋅i+b)
Формат входных данных

В первой строке заданы два числа n, q(1≤n≤2⋅105,1≤q≤5⋅105).

Во второй строке задан массив a(0≤ai≤109).

Следующие q строк содержат запросы в заданном формате.
Гарантируется, что будет хотя бы один запрос типа ?.

Формат выходных данных

Для каждого запроса типа ? выведите ответ в отдельной строке.
